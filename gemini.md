# Project Context: Clinkar

## Overview
This project is a Next.js 16 application named "clinkar". Only one workspace is active.

## Tech Stack
- **Framework**: Next.js 16.1.4 (React 19)
- **Database/Backend**: Supabase (SSR, JS client)
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript

## Directory Structure
- `app/src/app`: Application routes and pages.
- `app/src/components`: Reusable UI components.
- `app/src/data`: Static data files.
- `app/src/lib`: Utility functions and shared logic.
- `app/src/services`: Service layer for backend interactions.
- `app/src/scripts`: Standalone scripts.

## Key Dependencies
- `lucide-react`: Icons
- `stripe`: Payments
- `qrcode.react`: QR Code generation
- `uuid`: UUID generation

## Notes
- The project uses the Next.js App Router.
- Tailwind 4 is configured with `@tailwindcss/postcss`.
